Criando Sites com HTML e CSS

Aprendendo HTML

HTML básico

Aprenda os fundamentos do HTML, a linguagem de marcação usada para estruturar páginas web. Entenda como usar tags, inserir imagens, links, listas e tabelas de forma simples e prática.

HTML intermediário

Aprofunde seus conhecimentos em HTML com formulários, iframes, elementos semânticos e melhores práticas para organização do código. Ideal para quem já conhece o básico e quer evoluir.

Aprendendo CSS

CSS básico

Descubra como estilizar suas páginas com CSS. Aprenda a alterar cores, fontes, tamanhos, margens, bordas e alinhar elementos de forma visualmente agradável e responsiva.

CSS intermediário

Domine seletores mais avançados, pseudo-classes, posicionamento de elementos e layouts com Flexbox e Grid. Torne seu site muito mais dinâmico e bem estruturado.

CSS Avançado

Eleve o nível do seu design com animações, transições, variáveis CSS, media queries e técnicas modernas de responsividade. Deixe seu site com aparência profissional.
